# Library Manager

A small JavaScript module for managing a simple in-memory book catalog.

It provides functions to:

- List books in the format `"title by author"`
- Get all book summaries
- Find books by a specific author
- Calculate the total number of pages in the catalog

---

## Requirements

- Node.js (v16+ recommended)
- npm
- Jest (for testing)

---

## Installation

1. Initialize a Node project:

```bash
npm install
npm init -y
```

### Running Tests

Make sure Jest is installed, then run:

npm test
